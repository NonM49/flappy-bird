# Flappy Bird Clone (Pygame)

A simple Flappy Bird–style game built using Python and Pygame.
The player controls a bird that must fly between pipes without colliding. The goal is to survive as long as possible and achieve the highest score.

## Gameplay

The player presses a key to make the bird jump while gravity pulls it downward. Pipes move toward the player, and the player must time their jumps to pass through the gaps.

## Features

- Smooth player movement with gravity and jumping

- Randomly generated pipe obstacles

- Score system

- Collision detection (pipes and ground)

- Sound effects for jump, score, and game over

- Background and tile graphics

- Simple game loop and physics

## Tech Stack

### Language

- Python

### Library

- Pygame

## How to Run

-  Clone the repository

git clone https://github.com/yourusername/flappy-bird.git

- Navigate to the project folder

cd flappy-bird

- Install dependencies

pip install pygame

- Run the game

python main.py

## Controls

- SPACE  → Jump

- R  → Restart game

Avoid the pipes and try to get the highest score possible.
